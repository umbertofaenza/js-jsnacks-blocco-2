# JSNACK 1

## Traccia

Inserisci un numero, se è pari stampa il numero, se è dispari stampa il numero successivo.

## Svolgimento

- Con un prompt, chiedo all'utente un numero

- **SE** il numero inserito è pari

  - Stampo il numero

  **ALTRIMENTI**

  - Aggiungo 1 al numero inserito
  - Stampo il numero
